# 観測ログ運用 Runbook v1

この文書は Codex-CLImutipule プロジェクトにおける観測ログ（OpenTelemetry, JSONL）の収集・ローテーション・確認手順を整理するためのプレースホルダーです。

- Phase4 完了時に、otel_config.yaml の更新内容と Collector 起動手順を記載すること。
- `data/logs/current/observability/` 以下に保存するログ種別・命名規則・ローテーション条件を明文化すること。
- REPORT_DRAFT の更新手順とレビューフローを追記すること。

準備が整い次第、具体的な手順に差し替えてください。
